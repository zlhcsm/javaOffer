# 荷兰国旗问题
>*注意：此问题和外部描述虽然不一致，但都是一类问题*  
[测试地址：leetcode 75. 颜色分类](https://leetcode-cn.com/problems/sort-colors/)

### 题目描述
给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。

注意:
不能使用代码库中的排序函数来解决这道题。
### 示例
```
输入: [2,0,2,1,1,0]
输出: [0,0,1,1,2,2]
```
